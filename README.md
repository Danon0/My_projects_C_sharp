# Программа по описанию марки машины с параметрами
Написать программу, которая, получая на вход марку автомобиля от пользователя, выводит описание марки с параметрами.
Пример:
Программа: Введите марку автомобиля или done для остановки ввода : 
User: Tesla 
Программа: «Tesla: electrical car with автоматической коробкой передач, 5 местами, Андроид на борту»
Введите марку автомобиля или done для остановки ввода :
User:
Программа: «описание авто»
И тд

В Вашем приложении вы должны реализовать:
Фабрику, которая возвращает ICar по переданному enum CarType. Должен быть метод string GetDescription();
Интерфейсы электрокара/обычной машины IElectric, IMechanical
Интерфейсы коробки передач IMechanical, IAutomatical
ACar - абстрактный клас с реализацией бОльшого кол-методов 
Каждый автомобиль должна быть представлен отдельным классом.
При необходимости вывод информации можно делать более детальным, глубина наследований может быть глубже, чем ICar-ACar-AutomaticCar-ConcreteCar
