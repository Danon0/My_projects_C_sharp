# TaskHub
## Консольное приложение для управления задачами: создание, редактирование, поиск, выполнение, сохранение в файл и асинхронная работа со статистикой. Приложение работает через консольное меню.

### Функциональные возможности:

1. **Создание задачи**  
   - название;
   - описание;
   - приоритет (`Low` / `Medium` / `High`);
   - дедлайн;
   - статус (`New` / `InProgress` / `Done`).

2. **Просмотр задач**  
   - все задачи;  
   - выполненные;  
   - невыполненные;  
   - задачи с высоким приоритетом.

3. **Редактирование задачи**
   - название;
   - описание;
   - приоритет;
   - статус;

5. **Удаление задачи**

6. **Поиск задач**  
   - по названию (частичное совпадение);  
   - по статусу;  
   - по приоритету;

7. **Статистика**  
   - общее количество задач;  
   - количество выполненных;  
   - количество просроченных;  
   - распределение по приоритетам.

8. **Сохранение задач в файл**

9. **Загрузка задач из файла**

10. **Фоновая проверка дедлайнов**  
   - отдельный поток раз в 5 секунд проверяет просроченные задачи и выводит уведомление.
